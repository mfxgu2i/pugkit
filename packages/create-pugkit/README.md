## About

[pugkit](https://github.com/mfxgu2i/pugkit)の静的サイト制作環境をセットアップします。

## How To Use

### Interactive

```sh
npm create pugkit@latest
```

### With project name

プロジェクト名を指定して作成します。

```sh
npm create pugkit@latest my-pugkit-project
```

### Current directory

カレントディレクトリに作成します。

```sh
npm create pugkit@latest .
```

## Next Steps

プロジェクト作成後の手順：

```sh
npm install
npm run build
npm run start
```
